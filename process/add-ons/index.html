Add-ons

Jan 21st, 2013

The goal Engine Yard was planning to launch a new feature which gives customers the ability to add third-party software to their applications. The development of the API for this feature was already underway when I first joined the team who was working on it. I came in to help create a clear user interface for customers to activate add-ons for their applications. The sign up My plan was to address add-on activation in its four steps: 1. Sign up for an add-on. 2. Select which application to use it on. 3. Make configuration file changes in the application’s code. 4. Deploy application.
